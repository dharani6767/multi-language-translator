# Multi-Language Translator

A simple and intuitive web-based application for translating text across multiple languages. This project provides a user-friendly interface built with HTML, CSS, and JavaScript to help users communicate across language barriers.

## Features

- **Multi-Language Support**: Translate text between multiple supported languages
- **Real-Time Translation**: Instant translation feedback as you type
- **User-Friendly Interface**: Clean and intuitive design for easy navigation
- **Copy Functionality**: Quick copy-to-clipboard feature for translated text
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **No Installation Required**: Runs directly in your web browser

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- Internet connection (for API calls if using an external translation service)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/dharani6767/multi-language-translator.git
cd multi-language-translator
```

2. Open `index.html` in your web browser:
   - Simply double-click the `index.html` file, or
   - Right-click and select "Open with" your preferred browser

### Usage

1. Select the source language from the first dropdown
2. Select the target language from the second dropdown
3. Enter the text you want to translate in the input field
4. The translation will appear in the output field
5. Click the copy button to copy the translated text to your clipboard
